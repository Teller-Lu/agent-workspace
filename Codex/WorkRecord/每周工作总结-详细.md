# 每周工作总结（详细档）

> 本文件是"工作内容"的**详细档**（述职盘点用）。每天由 `Automation/daily_worklog.sh` 自动**起草**：读当天 git 变更 + 当天完成的正式待办 → 生成当天工作内容。
> **只补缺失、不覆盖**——终稿归人（你改了或补了，自动不再动）。简洁档见同目录 `每周工作总结.md`。
> **排列**：周作为一级标题 `#`，最新一周在最上面（倒序）。周内按天：周一~周五各一段、周末（周六+周日合并）一段。
> **线下会议/口头事项**若未留痕，自动抓不到，请人工补一段。
