# WorkRecord — 工作记录

工作区的工作总结与计划集中存放，与 `Automation/` 的每日自动化配合。

| 文件              | 谁维护         | 说明                                   |
| --------------- | ----------- | ------------------------------------ |
| `每周工作总结.md`     | 自动起草 + 人工定稿 | 简洁档（周报口径），`daily_worklog.sh` 每天补缺失天行 |
| `每周工作总结-详细.md`  | 自动起草 + 人工定稿 | 详细档（述职口径）                            |


**自动化写入规则**：只补本周缺失的天行，绝不覆盖已存在行（护人工精修）。漏跑次日回看自愈。
