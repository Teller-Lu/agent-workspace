# Sample — 示例项目说明

本目录是 agent-workspace 的**示例项目**，给后来者参考"一个典型项目该长什么样"。

## 最小集

- `AGENTS.md` — 项目入口（L1）：项目信息、规则、文件构造说明
- `待办.md` — 标准待办模板（待办总表 + 已完成），编号 `SP-NNN`

## 按需扩展

- `README.md` — 给人看的项目说明
- `需求.md` / `设计.md` — 需求清单与设计方案（开发中 L2 / 已完成 L1）
- `src/` 或 `Scripts/` — 代码与脚本

## 如何用它

1. 复制 `Sample/` 改名为你的项目（如 `MyProject/`）。
2. 把 `待办.md` 的示例条目替换为真实待办，编号改成你的缩写（在根级 `CLAUDE.md` 中登记）。
3. 在根级 `CLAUDE.md` 子项目索引中补一行。
4. 删除或保留本示例项目均可——保留则根级 `待办.md`「项目待办」会聚合它的条目（演示效果）。
